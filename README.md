# AwsIotCoreMqttBase
This example shows how to connect a NetBurner device to the AWS IoT Core service via MQTT by subscribing to two different topics, sending messages on one,  and receiving them on the other.

For a full explanation on how to use this repo, and for a step by step guide to get up and running, please see our article at:

https://www.netburner.com/learn/how-to-connect-your-embedded-system-to-aws-iot-core-with-mqtt/
